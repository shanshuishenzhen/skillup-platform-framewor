#### 登录演示 (2分钟)
```
首先，让我们登录管理后台。

1. 打开浏览器，访问 https://admin.skillup.com
2. 输入管理员邮箱和密码
3. 如果启用了人脸识别，请对准摄像头完成验证
4. 输入短信验证码（如果启用了双因素认证）
5. 成功登录后，我们来到了管理仪表板

注意：请确保在安全的网络环境下登录，不要在公共设备上保存密码。
```

#### 界面介绍 (4分钟)
```
现在我们看到的是管理仪表板，让我介绍一下主要区域：

1. 顶部导航栏

   - 学习资源：文件上传、资源库管理

3. 主内容区域
   - 显示当前选中功能的详细内容
   - 支持多标签页操作

4. 仪表板数据卡片
   - 用户总数：显示平台注册用户数量
   - 活跃用户：显示最近7天活跃用户
   - 课程总数：显示平台课程数量

```

#### 快速导航 (2分钟)
```
让我演示一些快速导航技巧：

1. 使用搜索框
   - 输入用户邮箱可以快速找到用户
   - 输入课程名称可以快速定位课程

2. 收藏常用功能
   - 点击菜单项旁边的星号图标
   - 收藏的功能会显示在顶部快捷栏

3. 键盘快捷键

   - Ctrl+1：切换到仪表板
   - Ctrl+2：切换到用户管理
```

### 第二集：用户管理详解 (15分钟)

#### 开场回顾 (1分钟)
```
欢迎回到 SkillUp Platform 管理员培训课程。
上一集我们学习了管理后台的基本操作，
这一集我们将深入学习用户管理功能。

用户管理是管理员最常用的功能之一，包括：
- 查看用户列表
- 批量导入用户
- 编辑用户信息
- 管理用户权限
```

#### 用户列表操作 (3分钟)
```
首先，让我们点击左侧菜单的"用户管理"。

在用户列表页面，我们可以看到：

1. 用户统计信息
   - 总用户数、活跃用户数、新注册用户数

2. 筛选和搜索
   - 按部门筛选：选择特定部门查看用户
   - 按角色筛选：学生、教师、管理员
   - 按状态筛选：活跃、禁用、待激活
   - 搜索框：输入姓名或邮箱进行搜索

3. 用户列表
   - 显示用户头像、姓名、邮箱、部门、角色、状态
   - 最后登录时间、注册时间

4. 批量操作
   - 选择多个用户进行批量操作
   - 批量发送邮件、批量修改状态
```

#### 批量导入用户 (6分钟)
```
现在我们学习最重要的功能：批量导入用户。

第一步：下载导入模板
1. 点击"批量导入"按钮
2. 点击"下载模板"链接
3. 保存Excel模板文件到本地

第二步：填写用户数据
让我们打开下载的模板文件：
- name：用户姓名（必填）
- email：邮箱地址（必填，不能重复）
- phone：手机号码（可选）
- department：所属部门（可选）
- position：职位（可选）
- employee_id：员工编号（可选）
- role：用户角色（student/teacher/admin）
- status：账户状态（active/inactive）

注意事项：
- 邮箱必须是有效格式
- 手机号请包含国家代码，如+86
- 角色默认为student
- 状态默认为active

第三步：上传文件
1. 回到批量导入页面
2. 点击"选择文件"按钮
3. 选择填写好的Excel文件
4. 系统会显示数据预览

第四步：设置导入选项
- 发送欢迎邮件：是否给新用户发送欢迎邮件
- 自动生成密码：系统自动生成安全密码
- 立即激活：用户可以立即登录

第五步：执行导入
1. 检查预览数据是否正确
2. 点击"开始导入"按钮
3. 等待导入完成
4. 查看导入结果报告

导入结果会显示：
- 成功导入的用户数量
- 失败的用户及失败原因
- 重复的用户处理方式
```

#### 单个用户管理 (3分钟)
```
除了批量导入，我们也可以单独管理用户：

1. 添加新用户
   - 点击"添加用户"按钮
   - 填写用户基本信息
   - 设置角色和权限
   - 选择是否发送激活邮件

2. 编辑用户信息
   - 在用户列表中点击"编辑"按钮
   - 修改用户资料
   - 更改用户角色
   - 保存修改

3. 重置用户密码
   - 点击用户操作菜单中的"重置密码"
   - 选择发送重置邮件或设置临时密码
   - 通知用户及时修改密码

4. 禁用/启用用户
   - 临时禁用：用户无法登录但数据保留
   - 重新启用：恢复用户正常访问权限
   - 永久删除：谨慎操作，会删除所有相关数据
```

#### 权限管理 (2分钟)
```
SkillUp Platform 有三种用户角色：

1. 学生 (Student)
   - 可以学习课程
   - 查看学习进度
   - 参与讨论和测验

2. 教师 (Teacher)
   - 学生的所有权限
   - 创建和编辑课程
   - 查看学生学习数据
   - 批改作业和测验

3. 管理员 (Admin)
   - 教师的所有权限
   - 用户管理
   - 系统配置
   - 数据统计和导出

权限修改：
- 在用户编辑页面选择新角色
- 权限变更会立即生效
- 建议谨慎分配管理员权限
```

#### 小结和下集预告 (1分钟)
```
这一集我们学习了用户管理的核心功能：
- 用户列表查看和筛选
- 批量导入用户的完整流程
- 单个用户的增删改操作
- 用户权限管理

下一集我们将学习课程管理，包括如何创建课程和上传学习资源。
感谢观看！
```

### 第三集：课程管理和资源上传 (20分钟)

#### 开场 (1分钟)
```
欢迎来到第三集：课程管理和资源上传。
这是管理员的另一个核心功能，我们将学习：
- 创建和编辑课程
- 上传各种学习资源
- 管理课程内容结构
- 发布和维护课程
```

#### 创建课程 (5分钟)
```
让我们从创建一个新课程开始：

1. 进入课程管理
   - 点击左侧菜单"课程管理"
   - 点击"创建课程"按钮

2. 填写基本信息
   - 课程标题：如"JavaScript从入门到精通"
   - 课程描述：详细介绍课程内容和目标
   - 课程分类：编程、设计、营销等
   - 难度级别：初级、中级、高级
   - 预计学习时长：以分钟为单位

3. 设置课程封面
   - 点击上传区域选择图片
   - 建议尺寸：1200x675像素
   - 支持JPG、PNG格式
   - 文件大小不超过5MB

4. 配置课程属性
   - 是否公开：公开课程所有人可见
   - 是否免费：设置课程价格
   - 学习期限：课程有效期设置
   - 证书设置：完成后是否颁发证书

5. 保存草稿
   - 点击"保存草稿"按钮
   - 课程创建成功，状态为"草稿"
```

#### 课程结构设计 (4分钟)
```
课程创建后，我们需要设计课程结构：

1. 添加章节
   - 在课程编辑页面点击"添加章节"
   - 输入章节名称：如"第一章：JavaScript基础"
   - 添加章节描述
   - 设置章节顺序

2. 添加课时
   - 在章节下点击"添加课时"
   - 选择课时类型：
     * 视频课时：上传视频文件
     * 文档课时：上传PDF、PPT等文档
     * 测验课时：创建在线测验
     * 作业课时：布置课后作业

3. 设置课时内容
   - 课时标题：如"1.1 什么是JavaScript"
   - 课时描述：简要说明课时内容
   - 预计时长：学习所需时间
   - 是否免费试看：吸引用户注册

4. 调整顺序
   - 拖拽章节和课时调整顺序
   - 使用上下箭头微调位置
   - 确保逻辑顺序合理
```

#### 视频文件上传 (6分钟)
```
现在我们学习如何上传视频文件：

1. 支持的视频格式
   - 推荐格式：MP4 (H.264编码)
   - 支持格式：AVI, MOV, WMV, FLV, WebM
   - 最大文件大小：2GB
   - 推荐分辨率：1920x1080

2. 上传步骤
   - 在课时编辑页面选择"视频"类型
   - 点击"上传视频"按钮
   - 选择本地视频文件
   - 等待上传进度完成

3. 视频信息设置
   - 视频标题：自动从文件名获取，可修改
   - 视频描述：添加详细说明
   - 标签设置：便于搜索和分类
   - 缩略图：系统自动生成或手动上传

4. 高级设置
   - 字幕文件：上传SRT格式字幕
   - 播放速度：允许用户调节播放速度
   - 下载权限：是否允许用户下载
   - 水印设置：添加版权水印

5. 视频处理
   - 上传完成后系统自动转码
   - 生成多种清晰度版本
   - 创建视频预览图
   - 处理完成后可以预览

注意：大文件上传可能需要较长时间，请保持网络稳定。
```

#### 其他资源上传 (3分钟)
```
除了视频，我们还可以上传其他类型的学习资源：

1. 文档资源
   - 支持格式：PDF, DOC, DOCX, PPT, PPTX
   - 最大大小：100MB
   - 自动生成预览图
   - 支持在线阅读

2. 音频资源
   - 支持格式：MP3, WAV, AAC, FLAC
   - 最大大小：500MB
   - 自动生成波形图
   - 支持章节标记

3. 图片资源
   - 支持格式：JPG, PNG, GIF, WebP
   - 最大大小：10MB
   - 自动压缩优化
   - 生成多种尺寸

4. 压缩包资源
   - 支持格式：ZIP, RAR
   - 包含代码示例、素材等
   - 学生可以下载学习

上传流程都类似：
- 选择文件类型
- 上传文件
- 填写资源信息
- 设置访问权限
- 保存并发布
```

#### 课程发布和管理 (1分钟)
```
课程内容准备完成后，我们可以发布课程：

1. 预览课程
   - 点击"预览"按钮
   - 以学生视角查看课程
   - 检查内容是否完整

2. 发布课程
   - 确认所有内容无误
   - 点击"发布课程"按钮
   - 课程状态变为"已发布"
   - 学生可以开始学习

3. 课程维护
   - 查看学习数据和反馈
   - 根据需要更新内容
   - 回复学生问题和评论
   - 定期优化课程质量
```

#### 小结 (1分钟)
```
这一集我们学习了：
- 创建课程的完整流程
- 设计合理的课程结构
- 上传各种类型的学习资源
- 发布和维护课程

下一集我们将学习数据监控和系统配置。
感谢观看！
```

#### 学习数据分析 (4分钟)
```
首先我们来看学习数据分析：

1. 进入数据统计页面
   - 点击左侧菜单"数据统计"
   - 选择"学习数据"标签

2. 用户学习统计
   - 学习进度分布：查看用户完成情况
   - 学习时长统计：总学习时间和平均时长
   - 活跃度分析：用户登录和学习频率
   - 成绩分布：测验和考试成绩统计

3. 课程数据分析
   - 课程热度排行：最受欢迎的课程
   - 完成率统计：各课程的完成率
   - 用户评分：课程评分和评论分析
   - 学习路径：用户的学习轨迹

4. 导出报表
   - 选择时间范围：最近7天、30天、自定义

这些数据可以帮助我们：
- 了解平台使用情况
- 优化课程内容
- 制定运营策略
- 提升用户体验
```

#### 系统监控 (3分钟)
```
接下来我们看系统监控功能：

1. 性能监控
   - CPU使用率：服务器处理器使用情况
   - 内存使用：系统内存占用状态
   - 磁盘空间：存储空间使用情况
   - 网络流量：数据传输统计

2. 应用监控
   - 响应时间：API接口响应速度
   - 错误率：系统错误发生频率
   - 并发用户：同时在线用户数量
   - 数据库性能：查询速度和连接数

3. 告警设置
   - 性能阈值：设置CPU、内存告警线
   - 错误告警：系统错误自动通知
   - 容量告警：存储空间不足提醒
   - 安全告警：异常登录行为检测

4. 日志查看
   - 系统日志：查看系统运行日志
   - 错误日志：查看错误详细信息
   - 用户操作日志：审计用户行为
   - API调用日志：接口调用记录

监控数据帮助我们：
- 及时发现系统问题
- 优化系统性能
- 保障服务稳定性
- 提升用户体验
```

#### 系统配置 (3分钟)
```
现在我们学习系统配置管理：

1. 基础设置
   - 平台名称：修改显示名称
   - Logo设置：上传平台Logo
   - 联系信息：设置客服联系方式
   - 版权信息：设置版权声明

2. 功能开关
   - 用户注册：开启/关闭自主注册
   - 人脸识别：启用/禁用人脸登录
   - 短信验证：配置短信验证功能
   - 邮件通知：设置邮件通知规则

3. 安全配置
   - 密码策略：设置密码复杂度要求
   - 登录限制：设置最大失败次数
   - 会话超时：设置会话过期时间
   - IP白名单：限制管理后台访问

4. 第三方集成
   - 存储服务：配置云存储服务
   - 邮件服务：配置SMTP服务器
   - 短信服务：配置短信发送接口
   - 支付服务：配置在线支付接口

配置修改后记得点击"保存"按钮。
```

#### 故障排除 (1分钟)
```
最后，我们简单介绍故障排除：

常见问题：
1. 用户无法登录
   - 检查账户状态是否正常
   - 确认密码是否正确
   - 查看是否被IP限制

2. 文件上传失败
   - 检查文件大小和格式
   - 确认网络连接稳定
   - 查看存储空间是否充足

3. 视频播放问题
   - 检查视频编码格式
   - 确认CDN配置正确
   - 测试网络速度

4. 系统响应缓慢
   - 查看服务器性能监控
   - 检查数据库连接
   - 清理系统缓存

遇到问题时：
- 先查看系统日志
- 联系技术支持
- 记录问题详情
- 及时反馈给开发团队
```
